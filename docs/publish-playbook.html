GitHub Pages Deployment Steps

How this site is published and how to troubleshoot workflow failures.

Publication flow

  • Push changes to docs/ on main.
  • Workflow .github/workflows/pages.yml uploads docs artifact.
  • GitHub deploys the artifact to Pages URL.

Troubleshooting

  • Check Settings -> Pages -> Source is GitHub Actions.
  • Check Actions logs for the Pages workflow run.
  • Verify docs files exist and commit reached main branch.
